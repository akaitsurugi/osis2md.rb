# osis2md.rb

Converts OSIS XML files into Markdown text.

This was created with a narrow scope: to be used with a custom, simplified [King James text](https://github.com/akaitsurugi/kingjesusbible) (the base for the King Jesus Bible) and the [Japanese Lifeline New Testament](https://github.com/akaitsurugi/japlifeline).

Not all OSIS features or tags are supported.
